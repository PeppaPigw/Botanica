# Changelog

## v1.0.0

- Added the offline-first garden, care tasks, reminder scheduling, snooze, skip, and completion flows.
- Added calendar history with care log dots, day agenda, filters, quick watering, and streak summaries.
- Added plant library, Discover search, localized species details, care defaults, tags, and expanded schema fields.
- Added photo journal, diary entries, share cards, compare tools, and offline media storage handling.
- Added Daily Flower ritual modes, deterministic daily content, saving, sharing, and AI note support.
- Added scan and identification flow with confidence, refinement chips, fallback browsing, and cached last result.
- Added Open-Meteo weather/environment support, local caching, unit preferences, and graceful offline fallback.
- Added multilingual support for English, Arabic, Spanish, and Chinese, including RTL-aware UI work.
- Added accessibility, release, store listing, app icon, splash, and pre-release QA documentation.
